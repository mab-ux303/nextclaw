# nextclaw

Your omnipotent personal assistant, OpenClaw-compatible and local-first, with CLI + built-in Web UI.

## Install

```bash
npm i -g nextclaw
```

## Quick start

```bash
nextclaw start
```

Then open `http://127.0.0.1:18791`.

## Common commands

```bash
nextclaw --version
nextclaw status
nextclaw stop
nextclaw update
```
