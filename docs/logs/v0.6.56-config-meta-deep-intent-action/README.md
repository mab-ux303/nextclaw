# v0.6.56-config-meta-deep-intent-action

## 迭代完成说明（改了什么）

本次通过 `/config-meta` 完成元规则升级，核心是把“深思模式下减少沟通成本”的要求固化为可执行机制：

1. 更新 `AGENTS.md` 的 `/config-meta` 指令说明：
   - 明确深思模式开启时，需要主动推理潜在意图；
   - 对高概率期望动作直接执行，而非反复确认。
2. 更新 Rulebook 规则 `config-meta-no-new-modules`：
   - 新增“显性意图 + 隐含意图”诊断要求；
   - 新增“高概率、低风险、可回滚动作应直接执行”的执行约束；
   - 补充了对应示例与反例。
3. 同步更新指令文档 `commands/commands.md` 的 `/config-meta` 条目，保持机制与索引一致。
