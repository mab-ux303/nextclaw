# Code Volume Metrics

该目录默认由 `pnpm metrics:loc` 维护，口径为源码 LOC，且不包含文档站 `apps/docs`。

- `latest.json`：最新一次快照（机器可读）
- `history.jsonl`：历史快照（每行一条 JSON）
- `comparison.json`：与 OpenClaw 的源码 LOC 对比结果（用于 README 对比徽章）
