# 对话能力

这页只讲“你能直接用起来的能力”，不要求先学命令行。

## 你能做什么

- 多轮会话：按会话持续上下文，不用每轮重述背景。
- 模型切换：可按会话指定模型，在速度和效果之间灵活切换。
- 技能加成：发送前可选择已安装 `skills`，让本轮回答遵循特定规则。
- 工具调用：可在对话中调用已启用工具（如搜索、网页抓取、任务执行）。
- 渠道一致：同一套核心对话能力可复用到本地 UI 与已接入渠道。
- 会话可管理：支持搜索、查看历史、清空、删除与重命名。

## 新手建议

1. 先在 UI 发起 1 个会话并完成 3~5 轮连续对话。
2. 再尝试切换一个模型，对比响应速度和质量。
3. 最后加一个 skill，体验“同问题不同策略”的效果。
