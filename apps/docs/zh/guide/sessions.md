# 会话管理

会话管理用于“让历史可查、上下文可控、问题可回溯”。

## 你可以做什么

- 列出会话并支持搜索、按渠道筛选
- 查看会话聊天历史
- 编辑会话元数据（`label`、`preferredModel`）
- 清空会话历史
- 删除会话

## 什么时候最有用

- 多渠道同时运行，想快速定位某条对话
- 某个会话上下文已混乱，需要清空重来
- 你想把长期会话按主题重命名和整理

## 新手建议

1. 给常用会话加 `label`，先建立最小命名规则。
2. 对异常会话优先“清空历史”再重试。
3. 定期删除无效会话，减少噪音。
